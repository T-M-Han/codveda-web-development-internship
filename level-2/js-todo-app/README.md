## Project Overview

TaskMate is a responsive JavaScript to-do list application that allows users to create and manage tasks. The project focuses on DOM manipulation, event handling, responsive layout design, and browser local storage.

Users can add tasks, mark tasks as completed, edit tasks, delete tasks, and filter tasks by status. The app also saves task data in the browser using local storage, so tasks remain available after refreshing the page.

## Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed
* Filter tasks by:

  * All
  * Active
  * Completed
* Task summary cards

  * Total tasks
  * Active tasks
  * Completed tasks
* Empty task message
* Form validation for empty input
* Browser local storage support
* Responsive layout for desktop, tablet, and mobile
