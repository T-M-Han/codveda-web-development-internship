## Project

### TaskMate — JavaScript To-Do List App

TaskMate is a responsive to-do list application built using HTML, CSS, and JavaScript. It allows users to add, edit, delete, complete, and filter tasks. It also uses browser local storage to save tasks after refreshing the page.

## Features Completed

* Responsive layout
* Flexbox layout
* CSS Grid summary cards
* Add task functionality
* Edit task functionality
* Delete task functionality
* Complete task functionality
* Filter tasks by all, active, and completed
* Local storage support
* Empty task message
* Form validation
