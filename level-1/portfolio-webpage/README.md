## Project Overview

This is a fictional personal portfolio website for **Alex Morgan**, a front-end developer. The project demonstrates basic web development skills using HTML, CSS, and JavaScript.

The portfolio includes sections for home, about, skills, projects, and contact. It also includes basic JavaScript interactivity such as a mobile navigation menu, project details modal, and contact form validation.

## Features

* Responsive portfolio webpage
* Navigation bar
* Hero section
* About section
* Skills section
* Project cards
* Contact form
* Mobile menu toggle
* Project details modal
* Basic form validation
* Smooth scrolling
* Responsive design for desktop, tablet, and mobile
